# 🔐 Security Policy

| Version | Supported |
| ------- | --------- |
| 1.0.x   | ✅         |
| < 1.0   | ❌         |

---

## 🚨 Reporting a Vulnerability

If you discover a potential security vulnerability, please **do not** report it through public GitHub issues.

### 📩 Contact

Send a detailed report to: **[rithika.ujjalsingh@gmail.com](mailto:rithika.ujjalsingh@gmail.com)**

---

## 🧾 Please include:

* A clear description of the vulnerability
* Steps to reproduce the issue
* Proof of concept (if applicable)
* Potential impact (Low / Medium / High)

---

## ⏱️ Response Timeline

* 📬 Acknowledgement within **48 hours**
* 🔄 Status update within **5–7 days**

---

## 🛡️ Responsible Disclosure Guidelines

To maintain a safe environment:

* Do **not** exploit vulnerabilities beyond proof-of-concept
* Do **not** access, modify, or delete data
* Allow time for fixes before public disclosure
